# Twilio Voice AI Workshop Web App

A self-contained workshop web app for guiding attendees through building a voice AI agent with Twilio ConversationRelay and Gemini Flash. The workshop supports both a Python FastAPI path and a Node.js Fastify path.

Open `index.html` directly in a browser, or serve the folder locally if you want normal `localhost` URLs:

```bash
npx serve .
```

No build step is required.

## What is included

- Six workshop chapters from setup through launch.
- Copyable Python, Node.js, terminal, TwiML, and prompt snippets.
- Runtime switcher for Python or Node.js workshop instructions.
- Local progress tracking with chapter badges.
- Interactive agent builder for name, persona, voice, Gemini model, tools, and handoff behavior.
- Light and dark themes.
